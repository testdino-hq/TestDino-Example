# Playwright Examples

Playwright examples for CI/CD providers supported by [TestDino](https://app.testdino.com/).

## Summary

| Example | Description |
| --- | --- |
| [AWS CodeBuild](./aws-codebuild/README.md) | Run Playwright tests in AWS CodeBuild and upload results to TestDino |
| [Azure DevOps](./azure-devops/README.md) | Run Playwright tests in Azure DevOps and upload results to TestDino |
| [CircleCI](./circleci/README.md) | Run Playwright tests in CircleCI and upload results to TestDino |
| [GitHub Actions](./github-actions/README.md) | Run Playwright tests in GitHub Actions and upload results to TestDino |
| [GitLab CI](./gitlab-ci/README.md) | Run Playwright tests in GitLab CI/CD and upload results to TestDino |
| [Jenkins](./jenkins/README.md) | Run Playwright tests in Jenkins and upload results to TestDino |
